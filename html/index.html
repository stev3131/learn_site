INTRODUZIONE

Cos'è la programmazione?

La programmazione è una fase che permette la creazione del software,
in parole semplici quando creariamo un applicazione per esempio app android
noi creiamo l'app che poi tutti potranno usufruire

Cos'è il linguaggio di programmazione?

E'una serie di istruzioni che scrivi nel tuo codice per produrre
l'ambito software,che avrà il compilatore,layout(per sciverci sopra)...

Cos'è un compilatore?

Innazitto tutti i linguaggi di programmazione hanno un compilatore
perchè il compilatore converte quello che scrivi nel linguaggio macchina
questo linguaggio macchina sarebbe la comunicazione con la CPU 01010101...

Come differenziamo i linguaggi per il loro uso

Linguaggi per siti web

Qui troviamo HTML,CSS,JavaScript,Node js,React js e php
Essi si dividono poi in front-end e back-end
Html e css saranno il front-end cioè il layout della pagina web
E i restanti lavoro dietro non sono visibile nel layout principale

Linguaggi per imparare per bene le basi

Qui troviamo C,C#,Python,Java,C++
Elenco solo questi perchè sono i più importanti
E sono molto potenti in quello che farete

Linguaggi per l'intelligenza artificale

Python per il machine learing
C e C++ per arduino
Bonus per lo sviluppo giochi 3D: C# per unity,C++ per unreal engine
